Timestamp Microservice API

Built with Node.js and Express.js

User Stories:

  1. The microservice will check if a Unix timestamp or natural language date string is passed
  2. If it is, it returns both the Unix timestamp and natural language form of that date
  3. If it does not contain a natural date string or Unix timestamp, it returns null for both properties

Example Input

https://timestamp-brendansweeny.c9users.io:8080/June%2023%202016
https://timestamp-brendansweeny.c9users.io:8080/1466640000

Example Output

{"unix":1466640000,"natural":"June 23, 2016"}
